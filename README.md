# Computational Methods & Tools (CE342L)

MATLAB implementations of numerical methods, optimization algorithms, interpolation techniques, and matrix computation methods completed as part of CE342L at Ghulam Ishaq Khan Institute (GIKI).

## Topics Covered

### Root Finding
- Bisection Method
- Fixed Point Iteration
- Newton-Raphson Method

### Interpolation
- Lagrange Interpolation

### Linear Systems
- Jacobi Method
- Gauss-Seidel Method

### Optimization
- Fletcher-Reeves Conjugate Gradient Method

### Eigenvalue & Matrix Methods
- Power Method
- Householder Tridiagonalization
